# API Testing with Postman 

## Initial Setup 

STEP 1: Import the Environment

1. Download the file TravelMates_Local.postman_environment.json
2. In Postman, click the gear icon ⚙️ (top right)
3. Click "Import"
4. Select the file and confirm
5. Close the window
6. In the dropdown at top right, select "TravelMates Local"

Now variables like {{base_url}} and {{auth_token}} will work!

STEP 2: Import the Collections

1. Click 'Import' (top left)
2. Select all the files in the '/postman/collecions' folder
3. Confirm the import
